N4 - 動詞ない形

~ ないでください

用法: 動詞ない形 + ないでください

解釋: 請求對方別做某些事

翻譯: (1) 請不要

例句: 美術館 びじゅつかん の 中 なか では、 写真 しゃしん を 撮 と らないでください。

翻譯: 美術館內請不要照相。

~ なければならない

用法: 動詞ない形／い形容詞去い + く／な形容詞／名詞 + で + なければならない

解釋: 表達某一行為是「義務」

翻譯: (1) 非...不可 (2) 不...不行 (3) 必須

例句: 勉強部屋 べんきょうへや の 電気 でんき は 明 あか るくなければなりません。

翻譯: 書房／讀書間的燈，一定要明亮。

~ なくてはいけない

用法: 動詞ない形／い形容詞去い + く／な形容詞／名詞 + で + なくてはいけない

解釋: 表達某一行為是「義務」

翻譯: (1) 非...不可 (2) 不...不行 (3) 必須

例句: 会議 かいぎ の 資料 しりょう をまとめなくてはいけないから、 先 さき に 帰 かえ って。

翻譯: 我必須要整理會議的資料，你先回去吧。

~ なくてもいい

用法: 動詞ない形／い形容詞去い + く／な形容詞+ で／名詞 + で + なくてはいけない

解釋: 表達某一行為/狀態是不必要的

翻譯: 不...也可以/無妨/沒關係

例句: このアプリを 使 つか って 電車 でんしゃ に 乗 の るので、 切符 きっぷ を 買 か わなくてもいいです。

翻譯: 用這個APP搭電車，所以不買票也可以

其它型態: (1) なくていい　(2) なくても構かまわない
